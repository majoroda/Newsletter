Introduction :

My site tells stories in different categories which includes recent sport , technology and entertainment news

And I implemented the grid structure system when developing the site

Inspiration :

My inspiration came from different sources;
  1. the site structure was inspired by news website e.g bbc.com
  2. the use of color was from various sport news websites e.g skysports.com

Accessibility :

  1. The site is operable in the sence that the web contents is keyboard accessible.
  2. It is understandable , the words and options are clear with no complex sentences for users with cognitive diability.
  3. It is robust , the texts and write ups can be easily read with ease by users with visual impairment.

Learning :

  1. Firstly, I visited different websites to get an idea on how the structure of the site should look like and what colors are mostly used.
  2. I reference w3schools frequently to grasp the idea of using a grid structure system
  3. Lastly, was implementing dynamic and also hover effects with vanilla javascript and css respectively

Evaluation I :

I think the implementation of the grid structure system of the webpages was succesful
because it was easier to make the site responsive on every screen size

Evaluation II :

Personally, with time I should be able to improve in adding dynamic effects on multiple pages with ease
